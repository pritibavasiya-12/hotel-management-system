# hotel-management-system
## 📌 Project Description
This is a web-based Hotel Management System developed using Python and Django. It allows users to book rooms, manage customer data, and track hotel operations efficiently.

## 🚀 Features
- User Registration & Login
- Room Booking System
- Customer Management
- Admin Dashboard
- Booking Status Tracking

## 🛠️ Technologies Used
- Python
- Django
- SQLite / MySQL
- HTML, CSS, Bootstrap

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run migrations:
   python manage.py migrate
4. Start server:
   python manage.py runserver
